Quadratische Gleichungen
Definition
Qudratische Gleichungen sehen ähnlich aus wie lineare Gleichungen, enthalten aber quadratische Rechenoperationen. \(ax\)2 \(bx\) \(c = 0\) Wenn nur eine Variable unbekannt ist, kann man diese berechnen.
Benötigtes Vorwissen
Für dieses Kapitel solltest du wissen, wie du lineare Gleichungen löst und umstellst.
Verschiedene Möglichkeiten
Es gibt mehrere Möglichkeiten qudratische Gleichungen zu lösen, bspw. , oder . Um diese zu verwenden muss sich die Gleichung die Normalform haben.
Faktorisieren
Man faktorisiert eine Gleichung, indem man die Gleichung in zwei Klammern schreibt. Man kann die Gleichung aus der Normalform so umformen, dass sie in einer Gleichung mit zwei Klammern steht, welche = 0 ist. Wenn ein Faktor als eine Klammer 0 ist, dann erhält man eine wahre Aussage. Dazu muss man die passende x-Wert welche sich leicht erkennen lassen einsetzen.
Beispiel: Die Gleichung wird mit dem Vorwissen über lineare Gleichungen in die Normalform gebracht. Es ist also auf einer Seite nur eine 0. \(x^2+5x+6 = 0\) Nun wird faktorisiert, dafür braucht man die Zahlen die addiert 5 und multipliziert 6 ergeben. Also in diesem Fall 2 und 3. Wenn das nicht möglich ist oder zu kompliziert sollte man eine andere Möglichkeit nutzen. \((x+2)(x+3) = 0\) Die Umstellung kommt so zu stande, dass wenn man die Klammer auflöst wieder auf deine nicht vereinfachte Normalform kommt. \((x\times x)+(x\times3)+(2\times x)\) + \((2\times3)\) Wenn man das wieder vereinfacht kommt man auf die Normalform. in der Gleichung mit den Klammern muss nun ein Faktor 0 sein, damit die Gleichung wahr ist, da alles 0 = 0. Also muss in einer Klammer 0 rauskommen. \((-2+2)(-2+3) = 0 \quad\vert\quad\)\((-3+2)(-3+3) = 0\) Wenn man nun beide möglichen Lösungen vereinfacht und zusammenfasst ist ein Faktor 0. \(0\times1 = 0\quad\vert\quad-1\times0 = 0\) Wenn man diese dann fertig ausrechnet erhält man zwei wahre Aussagen. \(0 = 0\) w.A.\(\quad\) | \(\quad 0 = 0\) w.A. Die Lösungsmenge enthält also zwei Zahlen. Man kann nun, wenn es gefordert ist, eine Probe machen. \((-3)^2+5\times(-3)+6 = 0\) Die Terme werden zuerst ausgerechnet. \(9-15+6 = 0\) Dann werden alle Terme addiert. \(0 = 0\) w.A. Nun wird die zweite Probe gemacht. \((-2)^2+5\times(-2)+6 = 0\) Die Terme werden wieder ausgerechnet. \(4-10+6 = 0\) Dann werden alle Terme addiert. \(0 = 0\) w.A. Nun wird die Lösungsmenge aufgeschrieben. \(L=\{-3;-2\}\) Mehrere Zahlen trennt man mit einem Semikolorn ( ; ), damit man mehrere Zahlen nicht mit Kommazahl verwechselt.

Die Mitternachtsformel (abc-Formel)
Die Mitternachtsformel ist eine Formel die zwar schwer zu merken ist, aber dafür auch bei komplexen Aufgaben funktioniert. Sie lautet: \(x = \frac{-b\pm\sqrt{b^2 - 4ac}}{2a}\) Das einfach ist, dass wenn man sie kennt nur einsetzten muss und dann ausrechnen. Einfacher zu merken ist es mit dem Reim: "Minus b, plus minus flott, unter der Wurzel steckt viel Schrott nämlich b Quadrat minus 4ac und das alles durch 2a, oje!" Es gibt mehrere Möglichkeiten, entweder unter der Wurzel ist eine positive Zahl, dann gibt es zwei Lösungen, eine 0, dann gibt es eine Lösung oder eine negative Zahl, dann gibt es ohne komplexe Zahlen keine Lösung.
Beispiel mit positiver Zahl unter der Wurzel: Hier kannst du wieder dein Vorwissen von den linearen Gleichungen anwenden, denn hier setzt man auch nur ein. \(3x^2\) \(4x\) \(1 = 0\) Nun nimmt man die sich allgemeine quadratische Gleichung dazu. \(ax^2\) \(bx\) \(c = 0\) Jetzt können die passenden Wert abgelesen werden. \(x = \frac{-b\pm\sqrt{b^2 - 4ac}}{2a}\) Diese werden dann in die Mitternachtsformel eingesetzt. \(x = \frac{-4\pm\sqrt{4^2 - 4 \times 3\times 1}}{2 \times 3}\) Nun rechnen wir aus, was unter der Wurzel steht, damit wir herausfinden, ob wir weiter rechnen müssen. \(x = \frac{-4\pm\sqrt{16 - 12}}{2 \times 3}\) Bei einer negativen Zahl unter der Wurzel müssten wir nicht weiterrechnen. \(x = \frac{-4\pm\sqrt{4}}{2 \times 3}\) Die Zahl ist positiv, also gibt es zwei Lösungen für x. \(x_1 = \frac{-4+2}{2 \times 3}\quad\) | \(\quad x_2 = \frac{-4 - 2}{2 \times 3}\) Beide Brüche werden zusammengefasst. \(x_1 = \frac{-2}{6}\quad\) | \(\quad x_2 = \frac{-6}{6}\)
Dann kann noch gekürzt werden.
\(x_1 = -\frac{1}{3}\quad\) | \(\quad x_2 = -1\) Hier ist es meistens sinnvoll Proben zu machen, da sich bei lösen einer komplexen Gleichung schnell Fehler einschleichen. \(3\times(-\frac{1}{3})^2 + 4\times(-\frac{1}{3})\) \(1 = 0\) Wir lösen zuerst die einzelnen Terme. \(3\times\frac{1}{9}+(-\frac{4}{3}) + 1 = 0\) Für die Übersichtlichkeit mach wir es in zwei Schritten. \(\frac{1}{3}-\frac{4}{3} + 1 = 0\) Nun addieren wir alle Terme zusammen. \(0 = 0\) w.A. Wir haben aber zwei Lösungen, also müssen wir auch zwei Proben machen. \(3\times(-1)^2 + 4\times(-1)+1 = 0\) Wir rechnen wiede die Terme aus. \(3+(-4)+1 = 0\) Nun addieren wir alle Terme zusammen. \(0 = 0\) w.A. Als letztes wird wieder eine Lösungsmenge angegeben. \(L=\{-1;-\frac{1}{3}\}\)
Nun kann es aber passieren, dass unter der Wurzel eine 0 herauskommt. Dann gibt es nur eine Lösung. Das heißt man braucht auch nur eine Probe.
Beispiel mit 0 unter dr Wurzel: \(4x^2\) \(4x\) \(1 = 0\) Wir setzen in die Mitternachtsformel ein. \(x = \frac{-b\pm\sqrt{b^2 - 4ac}}{2a}\) \(x = \frac{-4\pm\sqrt{4^2 - 4 \times 4\times 1}}{2 \times 4}\) Rechnen den Wert unter der Wurzel aus- \(x = \frac{-4\pm\sqrt{0}}{2 \times 4}\) Die Wurzel aus 0 kann nun ignoriert werden und feritg ausgerechnet werden. \(x = -\frac{1}{2}\) Nun macht man noch eine Probe. \(4\) \(-\frac{1}{2}^2\) \(4\) \(-\frac{1}{2}\) \(1 = 0\) Die Terme werden ausgerechnet. \(1\) \(-2\) \(1 = 0\) Dann werden alle Terme addiert. 0 = 0 w.A. Zum Schluss wird wie immer eine Lösungsmenge aufgeschrieben. \(L=\{\)\(\frac{1}{2}\}\)
Wenn unter der Wurzel eine 0 steht, dann gibt es keine Lösung. In diesem Fall ist die der Lösungsweg deutlich kürzer, da man nur den Teil unter der Wurzel ausrechnet, weshalb es sich immer lohnt diesen als erstes auszurechnen.
Beispiel mit negativer Zahl unter der Wurzel: \(2x^2\) \(-3x\) \(3 = 0\) Wir setzen in die Mitternachtsformel ein. \(x = \frac{-b\pm\sqrt{b^2 - 4ac}}{2a}\) \(x = \frac{-(-3)\pm\sqrt{-3^2 - 4 \times 2\times 3}}{2 \times 2}\) Nun rechnen wir aus, was unter der Wurzel steht. \(x = \frac{-(-3)\pm\sqrt{-15}}{2 \times 2}\) n.l. Da unter der Wurzel eine negative Wurzel steht schreibt man n.l. (nicht lösbar). \(L=\{\}\) Als Lösungsmenge gibt man nun eine leere Menge an.

p-q-Formel
Die p-q-Forml ist eine verinfachte Version der Mitternachtsformel. Sie lautet: \(x = -\frac{p}{2} \pm \sqrt{\frac{p}{2}^2-q}\) Sie ist zwar weniger komplex, ist aber darauf ausgelegt, dass vor dem x2 eine 1 steht also in der Gleichung nur x2. Die Normalform der quadratischen Gleichung ist also auch abgewandelt. \(x = x^2 + px + q = 0\) Man sieht, dass vor dem x2 keine Variable steht. Wenn vor dem x2 nun doch etwas steht muss man einen anderen Weg benutzen oder die Gleichung so umstellen, dass es passt. Genau wie bei der Mitternachtsformel gibt es wieder die drei Möglichkeiten einer positiven Zahl, einer 0 oder einer negativen Zahl unter der Wurzel. Diese Fäll verhalten sich genau so wie die der Mitternachtsformel
Beispiel mit positiver Zahl unter der Wurzel: \(2x^2 + 8x = 10\) Zuerst muss die Gleichung in die passende Normalform gebracht werden. Man stellt also um. \(2x^2 + 8x = 10\quad\vert-10\) Zuerst setzen wir die Gleichung = 0. \(2x^2 + 8x -10 = 10 - 10\) 10 und -10 gleichen sich aus, die rechte Seite ist also = 0. \(2x^2 + 8x -10 = 0\) Vor dem x2 steht aber noch eine Zahl, also müssen wir noch weiter umstellen. \(2x^2 + 8x -10 = 0\quad\vert\div2\) Dann passt die Gleichung auch für die p-q-Formel. \(x^2 + 4x -5 = 0\) Dann setzen wir in die p-q-Formel ein. \(x = x^2 + px + q = 0\) \(x = -\frac{p}{2} \pm \sqrt{\frac{p}{2}^2-q}\) \(x = -\frac{4}{2} \pm \sqrt{\frac{4}{2}^2-(-5)}\) Nun rechnen wir zuerst den Teil unter der Wurzel aus. Dieser wird auch Diskriminante gennant. \(x = -\frac{4}{2} \pm \sqrt{4+5}\) Die Zahl unter der Wurzel ist positiv, also haben gibt es zwei Lösungen. \(x = -\frac{4}{2} \pm \sqrt{9}\) Es gibt nun auch zwei x-Werte. \(x = -\frac{4}{2} + 3 \quad\vert\quad x = -\frac{4}{2} - 3\) Man kann beide Lösungen nun leicht berechnen. \(x_1 = 1 \quad\vert\quad x_2 = -5\) Eine Probe ist auch bei der p-q-Formel sinnvoll, ob man sie machen muss hängt normalerweise aber an der Aufgabenstellung. \(2 \times 1^2 + 8 \times 1 = 10\) Die Terme werden ausgerechnet. \(2+8=10\) Jetzt werden beide Seiten komplett ausgerechnet. \(10 = 10\) w.A. Genau wie bei der Mitternachtsformel haben wir zwei Lösungen, also auch zwei Proben. \(2 \times -5^2 + 8 \times -5 = 10\) Alle Terme werden wieder ausgerechnet. \(50 - 40 = 10\) Dann werden beide Seiten ausgerechnet. \(10 = 10\) w.A. Nun wird die Lösungsmenge aufgeschrieben, da wir wahre Aussagen haben. \(L=\{1;-5\}\)
Wie schon bei der Mitternachtsformel gibt es verschiedene Möglichkeit für die Zahl unter der Wurzel, welche einen Einfluss auf die Anzahl der Lösungen haben. Bei einer 0 unter der Wurzel gibt es eine Lösung.
Beispiel mit 0 unter der Wurzel: \(\frac{1}{2}x^2+\frac{9}{2}=-3x\) Die Gleichung muss = 0 gesetzt werden. \(\frac{1}{2}x^2+\frac{9}{2}=-3x\quad\vert +3x\) Die Kenntnisse vom Umstellen von Gleichungen sind wieder praktisch. \(\frac{1}{2}x^2+\frac{9}{2}+3x= 0\) Wegen des Kommutativgesetzes kann 3x so bewegen, dass es in die Normalform passt. \(\frac{1}{2}x^2+3x+\frac{9}{2}= 0\) Vor dem x2 steht \(\frac{1}{2}\), also müssen wir alles 2 rechenen, damit eine 1 vor dem x2steht und wir die p-q-Formel anwenden können. \(\frac{1}{2}x^2+3x+\frac{9}{2}= 0\quad\vert\times2\) Vor dem x2steht dann nämlich \(\frac{2}{2}\) was gekürzt 1 ist. \(x^2+6x+9=0\) Nun kann die p-q-Formel angewandt werden. \(x = x^2 + px + q = 0\) \(x = -\frac{p}{2} \pm \sqrt{\frac{p}{2}^2-q}\) \(x = -\frac{6}{2} \pm \sqrt{\frac{6}{2}^2-9}\) Zuerst wird wieder der Teil unter der Wurzel ausgerechnet. \(x = -\frac{6}{2} \pm \sqrt{9-9}\) \(x = -\frac{6}{2} \pm \sqrt{0}\) Unter der Wurzel steht eine 0 also gibt es eine Lösung für x. \(x = -\frac{6}{2}\) Es muss nur noch gekürzt werden. \(x = -3\) Es lohnt sich auch bei einer Lösung eine Probe durchzuführen. \(\frac{1}{2}\times -3^2+\frac{9}{2}=-3\times-3\) Die Terme werden zuerst ausgerechnet. \(\frac{9}{2}+\frac{9}{2}=9\) \(\frac{1}{2}\times 9+\frac{9}{2}=-3\times-3\) Dann werden beide Seiten ausgerechnet. \(9 = 9\) w.A. Zum wird wieder eine Lösungsmenge aufgeschrieben. \(L=\{-3\}\)
Die letze Möglichkeit ist wieder, dass unter der Wurzel eine negative Zahl steht. In diesem Fall gibt es keine Lösung und die Rechnung endet mit einem n.l. (nicht lösbar).
Beispiel mit 0 unter der Wurzel: \(x^2+2x+3=0\) Wenn die Gleichung schon in der Normalform ist muss nicht umgestellt werden und es kann direkt in die p-q-Formel eingesetzt werden. \(x = x^2 + px + q = 0\) \(x = -\frac{p}{2} \pm \sqrt{\frac{p}{2}^2-q}\) \(x=-\frac{2}{2}\pm\sqrt{\frac{2}{2}^2-3}\) Der Teil unter der Wurzel wird wieder zuerst ausgerechnet. \(x=-\frac{2}{2}\pm\sqrt{1-3}\) \(x=-\frac{2}{2}\pm\sqrt{-2}\) n.l. Der Teil unter der Wurzel ist negativ, somit ist die Gleichung nicht lösbar und es wird eine leere Menge als Lösungsmenge angegeben. \(L=\{\}\)

Zusammenfassung
Quadratische Gleichungen können auf viele Wege gelöst werden, bspw. Faktorisieren, wo die Gleichung auf zwei Klammern aufgeteilt wird und die Lösungsmengen mit der Nullregel bestimmt werden. Außerdem gibt es noch die Mitternachtsformel: \(x = \frac{-b\pm\sqrt{b^2 - 4ac}}{2a}\) und die p-q-Formel: \(x = -\frac{p}{2} \pm \sqrt{\frac{p}{2}^2-q}\) Die Mitternachtsformel ist eine Formel die zwar schwer zu merken ist, aber dafür auch bei komplexen Aufgaben funktioniert. Die p-q-Formel ist eine vereinfachte Version der Mitternachtsformel, die nur funktioniert, wenn die quadratische Zahl mit keiner anderen multipliziert wird. Man sollte bei den beiden Formel immer zuerst den Teil unter der Wurzel ausrechnen, dieser bestimmt, ob es zwei Lösungen (bei einer positiven Zahl unter der Wurzel), eine Lösung (bei einer 0 unter der Wurzel) oder gar keine Lösung (bei einer negativen Zahl unter der Wurzel) gibt. Wenn es keine Lösung gibt kann man das Lösen beenden. Zum Abschluss einer Aufgabe gibt man die Lösungsmenge an, welche alle möglichen Zahlen enthält die man für x einsetzen kann. Bei keiner Lösung ist es eine leere Menge (\(L=\{\}\)).
Faktorisieren: Lösung zur Aufgabe 1 \(-6x-3x^2-11 = 8\) Die Gleichung wird umgestellt, damit sie = 0 ist. \(-6x-3x^2-11 = -8\quad\vert+8\) \(-6x-3x^2-3 = 0\) Dann darf vor dem x2 keine Zahl stehen (also eine 1 die man weglassen kann). Außerdem, darf x2 auch kein Vorzeichen haben. \(-6x-3x^2-3 = 0\quad\vert\div(-3)\) \(2x+x^2+1 = 0\) Die Gleichung wird noch in die richtige Reihenfolge gebracht (das ist optional, hilft aber bei der Übersichtlichkeit). \(x^2+2x+1 = 0\) Nun wird faktorisiert. \((x+1)(x+1) = 0\) Nun wird die Nullregel angewandt. Da in beiden Klammern das gleiche steht, gibt es nur ein Ergebnis. \((-1+1)(-1+1) = 0\) Die Terme werden ausgerechnet. \(0\times0=0\) w.A. Nun wird die Lösungsmenge aufgeschrieben. \(L=\{-1\}\) Jetzt wird eine Probe gemacht. \(-6\times(-1)-3\times(-1)^2-11 = 8\) Die Terme werden ausgerechnet. \(6-3-11 = 8\) Dann werden die Terme zusammengefasst. \(8 = 8\) w.A. Die Lösungsmenge wird auch hier wieder angegeben. \(L=\{-1\}\)
Faktorisieren: Lösung zur Aufgabe 2 \(-x^2+3x-2 = 2x\) Die Gleichung wird umgestellt, damit sie = 0 ist. \(-x^2+3x-2 = 2x\quad\vert-2x\) \(-x^2+1x-2 = 0\) Dann darf x2 kein negatives Vorzeichen haben. \(-x^2+1x-2 = 0\quad\vert\times(-1)\) \(x^2-1x+2 = 0\) Man kann die Gleichung nicht faktorisieren, da es kein Zahlenpaar gibt, welches addiert -1 und multipliziert 2 ergibt. In diesem Fall ist das Faktorisieren nicht möglich. Man müsste eine andere Methode verwenden, um die Gleichung zu lösen.
Faktorisieren: Lösung zur Aufgabe 3 \(3x+2 = -x^2\) Die Gleichung wird umgestellt, damit sie = 0 ist. \(3x+2 = -x^2\quad\vert+x^2\) \(x^2+3x+2 = 0\) Nun wird faktorisiert. \((x+1)(x+2) = 0\) Nun wird die Nullregel angewandt. \((x_1+1) = 0\quad\vert\quad(x_2+2) = 0\) \(x_1 = -1\quad\vert\quad x_2 = -2\) Jetzt werden die Proben gemacht. \(3\times(-1)+2 = -(-1)^2\) Die Terme werden ausgerechnet. \(-3+2 = -1\) Dann werden die Terme zusammengefasst. \(-1 = -1\) w.A. Die andere Probe wird auch gemacht. \(3\times(-2)+2 = -(-2)^2\) Die Terme werden wieder ausgerechnet. \(-6+2 = -4\) Dann werden die Terme wieder zusammengefasst. \(-4 = -4\) w.A. Die Lösungsmenge wird angegeben. \(L=\{-1;-2\}\)
Mitternachtsformel: Lösung zur Aufgabe 1 \(2x^2+6=8x\) Die Gleichung wird umgestellt, damit sie = 0 ist. \(2x^2+6=8x\quad\vert-8x\) \(2x^2-8x+6=0\) Nun wird die Mitternachtsformel eingesetzt. \(x = \frac{-b\pm\sqrt{b^2 - 4ac}}{2a}\) \(x = \frac{-(-8)\pm\sqrt{(-8)^2 - 4 \times 2\times 6}}{2 \times 2}\) Der Teil unter der Wurzel wird zuerst ausgerechnet. \(x = \frac{8\pm\sqrt{64-48}}{4}\) Unter der Wurzel steht eine positive Zahl, es gibt also zwei Lösungen \(x = \frac{8\pm\sqrt{16}}{4}\) Nun wird die Wurzel ausgerechnet. \(x_1 = \frac{8+4}{4}\quad\vert\quad x_2 = \frac{8-4}{4}\) Nun wird alles ausgerechnet. \(x_1 = \frac{12}{4}\quad\vert\quad x_2 = \frac{4}{4}\) Es kann noch gekürzt werden. \(x_1 = 3\quad\vert\quad x_2 = 1\) Nun wird die Probe gemacht. \(2\times3^2+6=8\times3\) Die Terme werden ausgerechnet. \(18+6=24\) Dann werden die Terme zusammengefasst. \(24=24\) w.A. Die zweite Probe wird auch gemacht. \(2\times1^2+6=8\times1\) Die Terme werden wieder ausgerechnet. \(2+6=8\) Dann werden die Terme wieder zusammengefasst. \(8=8\) w.A. Die Lösungsmenge wird angegeben. \(L=\{1;3\}\)
Mitternachtsformel: Lösung zur Aufgabe 2 \(x^2+4x+1=2x\) Die Gleichung wird umgestellt, damit sie = 0 ist. \(x^2+4x+1=2x\quad\vert-2x\) \(x^2+2x+1=0\) Nun wird in die Mitternachtsformel eingesetzt. \(x = \frac{-b\pm\sqrt{b^2 - 4ac}}{2a}\) \(x = \frac{-2\pm\sqrt{2^2 - 4 \times 1\times 1}}{2 \times 1}\) Der Teil unter der Wurzel wird zuerst ausgerechnet. \(x = \frac{-2\pm\sqrt{4-4}}{2}\) Unter der Wurzel steht eine 0, es gibt also eine Lösung. \(x = \frac{-2\pm\sqrt{0}}{2}\) Die Wurzel kann nun ignoriert werden, da die Wurzel aus 0 = 0 ist. Es kann weiter ausgerechnet werden. \(x = \frac{-2}{2}\) Nun wird noch gekürzt. \(x = -1\) Nun wird die Probe gemacht. \((-1)^2+4\times(-1)+1=2\times(-1)\) Die Terme werden ausgerechnet. \(1-4+1=-2\) Dann werden die Terme zusammengefasst. \(-2=-2\) w.A. Die Lösungsmenge wird angegeben. \(L=\{-1\}\)

Die zweite Probe wird auch gemacht. \(3^2-4\times3-6=-3^2\) Die Terme werden wieder ausgerechnet. \(9-12-6=-9\) Dann werden die Terme wieder zusammengefasst. \(-9=-9\) w.A. Die Lösungsmenge wird angegeben. \(L=\{-1;3\}\)
p-q-Formel: Lösung zur Aufgabe 1 \(4x-x^2+10 = -11\) Zuerst wird das Kommutativgesetz genutzt, um die Terme zu tauschen. \(-x^2+4x+10 = 11\) Dann wird die Gleichung = 0 gesetzt. \(-x^2+4x+10 = -11\quad\vert+11\) Dann ist die Gleichung ber noch nicht in der Normalform für die p-q-Formel, da x2 ein negatives Vorzeichen hat. Das ändert man indem man (1) rechnet. \(-x^2+4x+21 = 0\quad\vert\times(-1)\) Nun steht vor dem x2 nichts mehr, aber die anderen Vorzeichen wurden auch verändert. \(x^2-4x-21 = 0\) Jetzt kann man in die p-q-Formel einsetzen. Zur Hilfe nochmal die p-q-Formel und die Normalform für die p-q-Formel. \(x = x^2 + px + q = 0\) \(x = -\frac{p}{2} \pm \sqrt{\frac{p}{2}^2-q}\) \(x = -\frac{-4}{2} \pm \sqrt{\frac{-4}{2}^2-(-21)}\) Der Teil unter der Wurzel wird ausgerechnet. \(x = -\frac{-4}{2} \pm \sqrt{2^2+21}\) \(x = -\frac{-4}{2} \pm \sqrt{25}\) Der Wert ist positiv, also gibt es zwei Lösungen und in zwei Wegen weiter ausgerechnet. \(x_1 = 2 + 5\quad\vert\quad x_2 = 2 - 5\) \(x_1 = 7\quad\vert\quad x_2 = -3\) Nun werden noch die Proben durchgeführt. \(4\times7-7^2+10 = -11\) Die Terme werden ausgerechnet. \(28-49+10 = -11\) Dann werden die Terme zusammengefasst. \(-11 = -11\) w.A. Die zweite Probe wird auch gemacht. \(4\times(-3)-(-3)^2+10 = -11\) Die Terme werden wirder ausgerechnet. \(-12-9+10 = -11\) Dann werden die Terme wieder zusammengefasst. \(-11 = -11\) w.A. Zum Schluss wird noch die Lösungsmenge angegeben. \(L=\{7;-3\}\)
p-q-Formel: Lösung zur Aufgabe 1 \(4x-x^2+10 = -11\) Zuerst wird das Kommutativgesetz genutzt, um die Terme zu tauschen. \(-x^2+4x+10 = 11\) Dann wird die Gleichung = 0 gesetzt. \(-x^2+4x+10 = -11\quad\vert+11\) Dann ist die Gleichung ber noch nicht in der Normalform für die p-q-Formel, da x2 ein negatives Vorzeichen hat. Das ändert man indem man (1) rechnet. \(-x^2+4x+21 = 0\quad\vert\times(-1)\) Nun steht vor dem x2 nichts mehr, aber die anderen Vorzeichen wurden auch verändert. \(x^2-4x-21 = 0\) Jetzt kann man in die p-q-Formel einsetzen. Zur Hilfe nochmal die p-q-Formel und die Normalform für die p-q-Formel. \(x = x^2 + px + q = 0\) \(x = -\frac{p}{2} \pm \sqrt{\frac{p}{2}^2-q}\) \(x = -\frac{-4}{2} \pm \sqrt{\frac{-4}{2}^2-(-21)}\) Der Teil unter der Wurzel wird ausgerechnet. \(x = -\frac{-4}{2} \pm \sqrt{2^2+21}\) \(x = -\frac{-4}{2} \pm \sqrt{25}\) Der Wert ist positiv, also gibt es zwei Lösungen und in zwei Wegen weiter ausgerechnet. \(x_1 = 2 + 5\quad\vert\quad x_2 = 2 - 5\) \(x_1 = 7\quad\vert\quad x_2 = -3\) Nun werden noch die Proben durchgeführt. \(4\times7-7^2+10 = -11\) Die Terme werden ausgerechnet. \(28-49+10 = -11\) Dann werden die Terme zusammengefasst. \(-11 = -11\) w.A. Die zweite Probe wird auch gemacht. \(4\times(-3)-(-3)^2+10 = -11\) Die Terme werden wirder ausgerechnet. \(-12-9+10 = -11\) Dann werden die Terme wieder zusammengefasst. \(-11 = -11\) w.A. Zum Schluss wird noch die Lösungsmenge angegeben. \(L=\{7;-3\}\)

p-q-Formel: Lösung zur Aufgabe 2 \(x^2-5x+9 = 3\) Die Gleichung wird umgestellt, damit sie = 0 ist. \(x^2-5x+9 = 3\quad\vert-3\) \(x^2-5x+6 = 0\) Nun wird in die p-q-Formel eingesetzt. \(x = -\frac{p}{2} \pm \sqrt{\frac{p}{2}^2-q}\) \(x = -\frac{-5}{2} \pm \sqrt{\frac{-5}{2}^2-6}\) Der Teil unter der Wurzel wird zuerst ausgerechnet. \(x = \frac{5}{2} \pm \sqrt{\frac{25}{4}-6}\) \(x = \frac{5}{2} \pm \sqrt{\frac{25}{4}-\frac{24}{4}}\) \(x = \frac{5}{2} \pm \sqrt{\frac{1}{4}}\) Die Zahl unter der Wurzel ist positiv, also gibt es zwei Lösungen. \(x_1 = \frac{5}{2} + \frac{1}{2}\quad\vert\quad x_2 = \frac{5}{2} - \frac{1}{2}\) Nun wird alles ausgerechnet. \(x_1 = \frac{6}{2}\quad\vert\quad x_2 = \frac{4}{2}\) Es kann noch gekürzt werden. \(x_1 = 3\quad\vert\quad x_2 = 2\) Nun wird die Probe gemacht. \(3^2-5\times3+9 = 3\) Die Terme werden ausgerechnet. \(9-15+9 = 3\) Dann werden die Terme zusammengefasst. \(3 = 3\) w.A. Die zweite Probe wird auch gemacht. \(2^2-5\times2+9 = 3\) Die Terme werden wieder ausgerechnet. \(4-10+9 = 3\) Dann werden die Terme wieder zusammengefasst. \(3 = 3\) w.A. Die Lösungsmenge wird angegeben. \(L=\{2;3\}\)
p-q-Formel: Lösung zur Aufgabe 3 \(x^2+4x+3=0\) Es wird in die p-q-Formel eingesetzt. \(x = -\frac{p}{2} \pm \sqrt{\frac{p}{2}^2-q}\) \(x = -\frac{4}{2} \pm \sqrt{\frac{4}{2}^2-3}\) Der Teil unter der Wurzel wird zuerst ausgerechnet. \(x = -2 \pm \sqrt{4-3}\) \(x = -2 \pm \sqrt{1}\) Die Zahl unter der Wurzel ist positiv, also gibt es zwei Lösungen. \(x_1 = -2 + 1\quad\vert\quad x_2 = -2 - 1\) Nun wird alles ausgerechnet. \(x_1 = -1\quad\vert\quad x_2 = -3\) Nun wird die Probe gemacht. \((-1)^2+4\times(-1)+3=0\) Die Terme werden ausgerechnet. \(1-4+3=0\) Dann werden die Terme zusammengefasst. \(0=0\) w.A. Die zweite Probe wird auch gemacht. \((-3)^2+4\times(-3)+3=0\) Die Terme werden wieder ausgerechnet. \(9-12+3=0\) Dann werden die Terme wieder zusammengefasst. \(0=0\) w.A. Die Lösungsmenge wird angegeben. \(L=\{-1;-3\}\)
